# Curriculum research

This directory will hold the evidence base used before exercise ideation begins.

Initial scope:

- framework: Plan d’études romand (PER);
- cycle: Cycle 2;
- HarmoS years: 6H, 7H, and 8H;
- disciplines: any foundational discipline with strong potential for short, interactive, feedback-rich learning.

Research should distinguish official curriculum statements from local interpretation. Do not place exercise ideas in this directory; curriculum evidence must remain useful even if every prototype is deleted.

## Subdirectories

- `sources/` — citation records for authoritative documents and pages
- `notes/` — structured synthesis by year, discipline, and objective
- `mappings/` — links between normalized PER objectives and future exercise IDs

Store links and quotations within their permitted use. Do not commit copyrighted source files unless redistribution is explicitly allowed.
