# Sources

Create one Markdown citation record per authoritative source. Use a stable ID such as `src-per-0001` and reuse that ID in exercise metadata.

Each record should include:

- stable source ID;
- title and publishing institution;
- canonical URL;
- document version or publication date when available;
- access date;
- covered cycle, years, domains, and objectives;
- relevant quotations with page or section references;
- reuse or copyright notes.

A source record is evidence, not a curriculum summary or an exercise proposal.
