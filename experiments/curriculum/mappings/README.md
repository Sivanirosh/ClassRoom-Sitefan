# Curriculum mappings

This directory will connect normalized curriculum objectives to exercise records once the paper catalog exists.

Mappings should make it possible to answer questions such as:

- Which designs address a given PER objective?
- Where does an idea progress from 6H to 8H?
- Which objectives have no proposed interactive treatment?
- Which exercises combine disciplines or representations?

Do not create mappings until both the cited objective and the exercise ID exist.
