# MES-01 calm game-mechanic precedents

## Question

Can a named commercial-game loop support a short, calm, no-writing graybox for repeating one invariant unit along a straight whole-unit span?

## Retained precedent: *A Monster’s Expedition*

### First-party source facts

- Draknek’s official press kit describes *A Monster’s Expedition* as a relaxing open-world puzzle adventure in which players progress by **pushing trees over to create pathways** between islands.
- In a first-party PlayStation Blog post, Draknek describes puzzle hints in terms of showing **where logs need to be placed to reach the next island**, then letting the player work backward.

These sources support only the broad source loop:

```text
push a tree/log → change the traversable world → reach another island
```

### Local adaptation hypothesis

The EX-0005 graybox proposes a narrower digital action:

```text
one discrete push → one complete fixed-distance roll
→ one same-length consequence mark → exact endpoint contact opens a crossing
```

The following are **not** source-backed facts and must remain explicit hypotheses:

- that one input can credibly represent one invariant unit displacement;
- that pupils will attend to equal displacement rather than tap until success;
- that accumulated action count will be understood as a length measure;
- that consequence marks make continuity visible without becoming a pre-counted ruler;
- that the adaptation is fun, age-appropriate, comprehensible in French, accessible in practice, or instructionally effective.

The graybox therefore shows no prospective slots, target numeral, ruler graduations, timing window, aim, swipe-distance control, written response, report, or score. Contact marks appear only after actions. Failure of pupils to connect the repeated action with the final unit-bearing count is a kill condition, not a prompt for cosmetic polish.

## Retained precedent: *Chinese Checkers*

### First-party source facts

Hasbro’s published rules describe two relevant move relations: moving a marble to an adjacent empty hole, and jumping over an adjacent marble into the empty hole directly beyond it. A jumped marble remains in place, and a turn may chain jumps.

These facts support the abstract relationship:

```text
adjacent move → one board interval
jump over an adjacent piece → two board intervals in one action
```

### Local adaptation hypothesis

EX-0004 replaces the source board with a true square unit grid and asks the pupil to predict a small integer before an automatic straight chain resolves. Each horizontal or vertical cell side is one invariant unit; a later visible obstacle forces one two-unit jump without removing the obstacle. Diagonal movement is deliberately deferred because a square diagonal has length `√2` relative to its side. The source does **not** establish length learning, integer prediction, automatic execution, orthogonal transfer, under/overshoot feedback, French comprehension, accessibility equivalence, or fun for this cohort.

The adaptation must be rejected if it becomes a higher/lower hidden-number game, if pupils count decorative pixels rather than use the explicit cell-side unit relation, or if action count is confused with length in unit-lengths.

## Copying boundary

For EX-0005, retain only the abstract push/roll-to-create-a-path relationship from *A Monster’s Expedition*. For EX-0004, retain only the adjacent-move / jump-over relationship from *Chinese Checkers*. Do not copy characters, boards, holes, islands, puzzle layouts, artwork, animation, writing, branded terminology, or presentation from either source.

## Evidence boundary

Commercial-game documentation establishes provenance for a source mechanic, not evidence of transfer to `MES-01`. Only owner replay and later pupil observation can judge game feel and semantic fit. Source provenance alone does not imply owner approval to expand.
