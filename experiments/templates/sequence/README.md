# Learning-sequence workflow templates

Use these templates after a roadmap structure is approved and one conceptual sequence is selected.

```text
LEARNING-BRIEF → MECHANIC-PITCHES → GRAYBOX-SCRIPT
→ one playable graybox → owner mechanic gate → full exercise DESIGN
```

Recommended working location:

```text
experiments/design/sequences/<sequence-id-lowercase>/
├── LEARNING-BRIEF.md
├── MECHANIC-PITCHES.md
└── GRAYBOX-SCRIPT.md
```

Copy and rename the templates; do not edit the templates in place.

The binding process is [`../../design/prototype-production-workflow.md`](../../design/prototype-production-workflow.md). Search [`../../game-mechanics/`](../../game-mechanics/) before doing new precedent research.
